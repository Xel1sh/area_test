Ход работы по проекту Демо (Вася и Лена)
- (Вася) делает форк от демо-проекта на гитхабе
- (Вася) приглашает участников (Лену)
- (Вася) настраивает обязательность PR для ветки master!!!
- (Лена) принимает приглашение

- (Вася) создает очередную задачу/проблему
- (Вася) выбирает задачу и создает для её решения ветку (feature/#-задача-1)
- (Вася) решает задачу в проекте, делает коммиты в ветку (в названии коммита - #<#issue>)
- (Вася) создает PR, назначает Лену рецензентом + метки (называть PR xxx)

- (Лена) делает ревью PR (pull/закачать/проверить), отправляет на доработку

- (Вася) дорабатывает задачу в проекте, перезапрашивает PR

- (Лена) делает ревью PR, одобряет слияние

- (Вася) делает слияние ветки с master
